# 🚀 Getting Started with LearningGuide

This guide walks you through setting up and running the app from scratch.

---

## Prerequisites

- **Python 3.10+** (Python 3.12 recommended)
- A terminal / command line
- Optional: A free API key from [Groq](https://console.groq.com) for AI features

Check your Python version:
```bash
python3 --version
```

---

## Step 1: Get the Code

```bash
git clone https://github.com/jacob-mazurkiewicz/LearningGuide.git
cd LearningGuide
```

---

## Step 2: Set Up the Web App

### Install Dependencies

```bash
cd web/backend
pip install -r requirements.txt
```

This installs:
- **FastAPI** — the web framework
- **Uvicorn** — the server that runs FastAPI
- **SQLAlchemy** — the database toolkit
- **Pydantic** — data validation
- And more (see `requirements.txt` for details)

### (Optional) Configure AI

```bash
cp .env.example .env
```

Open `.env` in your text editor and add your API key. The easiest option:

1. Go to [https://console.groq.com](https://console.groq.com)
2. Sign up for free
3. Create an API key
4. Add to `.env`: `GROQ_API_KEY=gsk_your_key_here`

### Start the Server

```bash
uvicorn main:app --reload
```

You'll see:
```
INFO: Uvicorn running on http://127.0.0.1:8000 (Press CTRL+C to quit)
INFO: Started reloader process
```

### Open the App

Go to **http://localhost:8000** in your browser. 🎉

## Step 3: Create Your First Learning Plan

### Manual Approach

1. Click the **+** button in the sidebar
2. Enter a plan title (e.g., "Learn Python in 8 Weeks")
3. Click **Save Plan**
4. Click **Add Goal** to add your first major goal
5. Inside the goal, add subtasks
6. Inside each subtask, add daily tasks
7. Check off tasks as you complete them!

### AI Approach

1. Click the green **✨** (stars) button in the sidebar
2. Type what you want to learn: *"Learn data science with Python in 6 weeks"*
3. Set your duration and daily commitment
4. Click **Generate Plan**
5. Review the generated plan
6. Click **Create This Plan** to save it

2. Find this section:

4. Refresh your browser — you'll see the change immediately!

## Troubleshooting

### "Address already in use" error
Another process is using port 8000. Either:
- Stop the other process, or
- Run on a different port: `uvicorn main:app --reload --port 8001`

### "Module not found" error
You're not in the right directory. Make sure you're in `web/backend/`:
```bash
cd /path/to/LearningGuide/web/backend
uvicorn main:app --reload
```

### AI generation fails
- Check that you've added a key to `.env`
- The file should be `web/backend/.env` (not `.env.example`)
- Restart the server after changing `.env`
